Radar Cidadão

Aplicação web desenvolvida a partir da API da Câmara dos Deputados, com foco em apresentar dados públicos de forma clara, organizada e acessível para diferentes perfis de usuários, como cidadãos, jornalistas e professores.

PHP HTML CSS JavaScript

Monitoramento da Poluição Urbana

Aplicação web que utiliza dados em tempo real via API para monitorar indicadores ambientais, como qualidade do ar, temperatura e clima, com foco também na conscientização dos usuários e registro de ações.

Radar Cidadão
Python Flask Bootstrap HTML CSS JavaScript

Contextualização

O acesso aos dados da Câmara dos Deputados pode ser complexo e pouco intuitivo para grande parte da população. Eleitores, professores e jornalistas enfrentam dificuldades para localizar e interpretar informações relevantes, muitas vezes dispersas e sem uma visualização estruturada ou neutra.

Objetivo

Desenvolver uma aplicação web que centralize e simplifique o acesso aos dados da Câmara dos Deputados, permitindo que usuários, mesmo sem conhecimento técnico, consigam consultar e compreender essas informações de forma prática e eficiente.

Metodologia Aplicada

O desenvolvimento do projeto foi conduzido utilizando a metodologia ágil Scrum, estruturada em três sprints, cada uma com duração de três semanas. Ao final de cada ciclo, foram realizadas sprint reviews, permitindo a análise das entregas, identificação de melhorias e ajustes para as próximas etapas do desenvolvimento.

Minha Atuação

Atuei como Product Owner (PO), sendo responsável pela comunicação com o cliente, definição de requisitos e organização das entregas do projeto. Também contribui na estruturação do repositório no GitHub e na documentação, além de participar da definição da organização e apresentação dos dados na interface.

Resultados obtidos

O projeto resultou em uma aplicação em desenvolvimento que já permite a visualização de dados da Câmara dos Deputados de forma mais simples e acessível, facilitando o entendimento de informações complexas por diferentes perfis de usuários.

Monitoramento da Poluição Urbana
HTML CSS JavaScript Bootstrap MYSQL

Contextualização

A obtenção de dados atualizados sobre a poluição urbana nem sempre é acessível ou clara para a população. Informações sobre qualidade do ar, água e outros indicadores ambientais costumam estar dispersas ou apresentadas de forma pouco intuitiva, dificultando o acompanhamento e a conscientização dos usuários.

Objetivo

Desenvolver uma aplicação web capaz de centralizar e apresentar dados ambientais em tempo real, como qualidade do ar e outros indicadores, de forma clara e acessível. O projeto também busca conscientizar os usuários sobre práticas que contribuam para a redução da poluição no dia a dia.

Metodologia Aplicada

O desenvolvimento foi conduzido com base na organização e estruturação da interface a partir de protótipos elaborados no Figma, seguidos da implementação do front-end com foco na apresentação clara e eficiente dos dados. A construção do projeto priorizou a usabilidade e a organização das informações para facilitar a compreensão dos usuários.

Minha Atuação

Atuei no desenvolvimento do front-end da aplicação, sendo responsável pela criação do design no Figma e pela implementação da interface. Também participei da definição da estrutura do site e da organização dos dados apresentados, buscando uma experiência clara e intuitiva para o usuário.

Resultados obtidos

O projeto resultou em uma aplicação funcional capaz de exibir dados ambientais em tempo real, com uma interface organizada e de fácil compreensão. A solução contribui para o acompanhamento da poluição urbana e incentiva a conscientização dos usuários sobre questões ambientais.
